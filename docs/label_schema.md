# Label schema

The dataset uses the International Academy of Cytology Yokohama System for breast FNAC reporting:

- C1: Insufficient or inadequate
- C2: Benign
- C3: Atypical
- C4: Suspicious for malignancy
- C5: Malignant

The released patch directory uses Roman numeral class folders:

- `I` = C1
- `II` = C2
- `III` = C3
- `IV` = C4
- `V` = C5

Patch labels are expert-verified reference labels for patch-wise cytomorphological classification. They should not be interpreted as final patient-level diagnoses.
